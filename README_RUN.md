# Running the OpenClaw Lead Discovery System

## Setup

Clone the repository:

git clone <repo-url>

cd openclaw-workspace

Create virtual environment:

python -m venv push-venv

Activate environment (PowerShell):

.\push-venv\Scripts\Activate.ps1

Install dependencies:

pip install -r requirements.txt


## Running OpenClaw

Start the OpenClaw CLI:

openclaw


## Example Commands

Find leads:

find tutoring centers in Glendale

Enrich:

enrich these

Rank:

rank these

Draft outreach:

write outreach for these

Export:

export this list
